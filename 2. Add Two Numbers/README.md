# 2. 两数相加

给定两个非空链表来表示两个非负整数。位数按照逆序方式存储，它们的每个节点只存储单个数字。将两数相加返回一个新的链表。

你可以假设除了数字 0 之外，这两个数字都不会以零开头。

**示例：**

> **输入：** (2 -> 4 -> 3) + (5 -> 6 -> 4)
>
> **输出：** 7 -> 0 -> 8
>
> **原因：** 342 + 465 = 807

# 解答

这道题不难，我们只需要依次将列表 `l1` 和 `l2` 对应位置的两个数相加即可。

唯一需要注意的是相加的结果可能会产生溢出，即产生一个进位。我们使用 `carryBit` 表示进位，如果相加结果小于 10，则无进位，否则有进位。
